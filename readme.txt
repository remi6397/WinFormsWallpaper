= Wowpaper

// vim: set ft=asciidoc :

Wowpaper is a self-contained application that can set any website as your desktop background (under desktop icons like ActiveDesktop, but fullscreen and non-interactive).

This application was tested and confirmed to work on Windows 10.

== Remixing

If you want to set your own application's window as a wallpaper, feel free to use the code from `WallpaperManager.cs`.
